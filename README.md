## **PikPak 是什麼？跟 OneDrive 根本不是同一種東西**

很多人第一次聽到 PikPak 的反應是「這是什麼牌子的雲端硬碟？」，然後就直接拿來跟 OneDrive、Google Drive 比較。

其實這樣比有點不太公平，因為兩個的設計邏輯根本不同。

**OneDrive** 的核心是「同步」。它的存在意義是把你電腦上的檔案同步備份到雲端，然後跨裝置可以存取。你在電腦裡新增一個 Word 檔，OneDrive 馬上幫你備份；你換到手機也能打開。如果你是 Microsoft 365 的訂閱戶，那 OneDrive 就是你整個工作環境的延伸。

**PikPak** 的核心則是「雲端下載 + 私密儲存」。它更像是一個放在遠端伺服器上的私人 NAS（網路附接儲存），你可以把各種來源的檔案直接下載進去——包括 BT 種子、磁力鏈接（Magnet）、HTTP 連結，甚至 Twitter、TikTok、Facebook 的影片連結，全部都能丟給 PikPak 的伺服器去下載，速度極快，然後你再從 PikPak 慢慢取用。

> 簡單說：OneDrive 是把你的東西「備份上去」，PikPak 是讓伺服器幫你「把東西抓下來存著」。

這兩個用途其實差很多，但如果你只是需要一個大容量、好用的地方存檔案、看影片，那 PikPak 確實可以取代 OneDrive 的位置，而且省下不少錢。
